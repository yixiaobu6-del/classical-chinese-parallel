# 文言文对照阅读器

文言文读不懂？
**对照阅读+词汇扩展帮你学。**

[![HTML](https://img.shields.io/badge/打开即用-无需安装-orange)]()

## 文言文不再"难读"

- **词汇扩展**：200+条词汇，含释义
- **学习追踪**：done/todo筛选+进度条
- **三种模式**：原文/对照/白话阅读模式
- **分类学习**：按难度/主题分组

## 怎么用

打开 `web/reader.html` → 选择文章 → 对照阅读
